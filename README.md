# AI Interview Preparation Platform

## Project Overview

The AI Interview Preparation Platform is a web-based application developed to help students and job seekers prepare for technical and HR interviews in an interactive way. The platform provides interview practice, AI-style feedback, voice interview support, and a chatbot interface similar to ChatGPT.

This project is designed to improve communication skills, technical knowledge, and interview confidence through continuous practice.

---

# How the Project Works

## 1. User Registration & Login

Users can create an account by entering:
- Full Name
- Email
- Password
- Target Role
- Experience Level

After successful registration or login, the user is redirected to the dashboard.

---

## 2. Dashboard

The dashboard displays:
- Total Interviews
- Total Answers
- Average Score
- Latest AI Feedback

Users can:
- Start AI Interview
- Open AI Coach Chatbot
- Logout from the system

---

## 3. AI Interview Module

The interview module automatically asks technical interview questions one by one.

Features:
- ChatGPT-style interface
- Automatic next question flow
- AI-style feedback generation
- Voice answer support using Speech Recognition API
- Text-based answer submission
- Real-time score tracking

The system evaluates the answer length and provides improvement suggestions.

---

## 4. Voice Interview Feature

Users can answer interview questions using voice input.

How it works:
- User clicks microphone button 🎤
- Browser speech recognition converts voice into text
- Converted text is automatically added to the answer box

---

## 5. AI Coach Chatbot

The chatbot allows users to ask:
- Java interview questions
- HR interview questions
- Coding doubts
- Career guidance questions

The chatbot works using REST API communication between React frontend and Spring Boot backend.
